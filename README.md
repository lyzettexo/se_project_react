# WTWR (What to Wear)

## Description

WTWR is a React app that displays current weather and recommends clothing items based on the weather type. Users are able to view clothing cards, open the item as a preview modal, and access a add-garment modal layout.

## Functionality

- Retrieves weather data from an API
- Displays current location and temperature
- Filters clothing items by weather type: hot, warm, or cold
- Responsive layout shows clothing cards
- Preview modal opens when the clothing item is clicked
- Ability to add a new garment via form modal

## Technologies Used

- React
- Vite
- JavaScript
- CSS
- HTML
- GitHub Pages
- OpenWeather API
